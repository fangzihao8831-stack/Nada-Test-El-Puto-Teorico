**dato.md — content:**

Type definition: Preguntas que piden un dato concreto (velocidad, distancia, plazo, tasa, puntos) contextualizado en una situacion practica. Representan aproximadamente 10% del examen DGT. La respuesta depende de conocer el valor exacto de la norma. Siempre contextualizados como aplicacion de la norma, nunca como trivia sobre la norma en si.

Level definitions (from CONTEXT.md, verbatim):
- Nivel 1 (Facil): "Un unico hecho o regla. Los distractores son claramente de otro contexto."
- Nivel 2 (Medio): "Requiere conocer un umbral especifico, excepcion, o condicion combinada. Un distractor es plausible para un lector descuidado."
- Nivel 3 (Dificil): "Todos los distractores son valores/reglas reales del temario aplicados a una condicion ligeramente distinta. No deducible por eliminacion."

Distractor rules for dato:
- Nivel 1: One distractor from a clearly different context (different vehicle type, different road type). One distractor that is obviously wrong (e.g., a value from a completely unrelated category).
- Nivel 2: At least one distractor is a value from the same vehicle/road type but a different condition (e.g., the same vehicle at a different road type, or an adjacent threshold value that applies to a different subcategory).
- Nivel 3: All three options are real values from the temario for the same scenario category. The correct option applies to the EXACT conditions stated. The wrong options apply to related but different conditions. NO monotonic sequence permitted — options must come from different conditions of the same scenario (NOT from adjacent rows of the same table row). Example from CONTEXT.md: 70/80/90 km/h for a remolque question — all real values but for different remolque conditions or road types.

For the Level 3 example, use the CONTEXT.md approved example (Circula por carretera convencional arrastrando un remolque de 600 kg de MMA — this is an approved example).

Add checklist:
```
## Checklist por nivel

### Nivel 1
- [ ] El distractor incorrecto proviene de un contexto claramente distinto (diferente tipo de vehiculo, diferente tipo de via)
- [ ] No hay ambiguedad sobre cual es la respuesta correcta conociendo la norma

### Nivel 2
- [ ] Al menos un distractor es un valor real del temario que aplica a una condicion distinta del mismo escenario
- [ ] Un alumno descuidado podria elegir el distractor plausible

### Nivel 3
- [ ] Los 3 valores son reales del temario (no inventados)
- [ ] Los 3 valores aplican a condiciones distintas del mismo escenario
- [ ] NO forman una secuencia monotonica de la misma fila de la tabla
- [ ] Un alumno sin dominio exacto de la norma no puede eliminar ninguna opcion por logica
```

**directo.md — content:**

Type definition: Preguntas que piden al alumno aplicar o reconocer una regla o norma directamente, sin escenario vehicular complejo. El enunciado plantea una situacion sencilla o una pregunta conceptual sobre obligaciones, prohibiciones o excepciones. Representan aproximadamente 25-30% del examen DGT.

Level definitions (same definitions as dato, 3 levels — "Un unico hecho...", "Requiere conocer un umbral...", "Todos los distractores...").

Distractor rules for directo:
- Nivel 1: Distractors are clearly from a different context. The correct answer is unambiguous to anyone who knows the rule.
- Nivel 2: At least one distractor represents a common exception or adjacent rule that a careless reader might confuse with the correct answer.
- Nivel 3: All three options are plausible rules that exist in the temario. The correct answer requires knowing the specific exception, condition, or which rule prevails. The student cannot eliminate any option through general knowledge alone.

Use the CONTEXT.md approved examples for directo (Facil: cinturon de seguridad; Medio: giro a la derecha en semaforo rojo; Dificil: adelantar en puente movil). These are approved reference examples.

Checklist similar to dato, adapted for directo.

**completar.md — content:**

Type definition: Preguntas que presentan una afirmacion incompleta que el alumno debe completar eligiendo la opcion correcta. El enunciado termina con suspension de puntos o una clausula abierta. Representan aproximadamente 20-25% del examen DGT.

Level definitions (3 levels, same as dato/directo):
- Nivel 1: Completion is unambiguous; the wrong options complete the sentence with clearly incorrect information.
- Nivel 2: The completion requires knowing a specific distance, time limit, or exception. At least one distractor completes the sentence with a value that could be confused by a careless reader.
- Nivel 3: All three completions produce grammatically and contextually plausible sentences. The correct completion requires knowing the precise rule. The wrong completions describe real but inapplicable rules.

Use CONTEXT.md approved examples for completar (Facil: senal de maniobra; Medio: distancia minima en cruce; Dificil: ciclistas en autovias). These are approved reference examples.

Distractor rules:
- Nivel 1: Wrong options complete the sentence with an obviously incorrect fact.
- Nivel 2: At least one wrong option uses a real value but from a different context or distance category.
- Nivel 3: All options produce sentences that sound like plausible regulations. A student must know the exact rule to identify the correct completion.

Checklist: same structure as dato.md checklist adapted for completar.

**situacional.md — content:**

Type definition: Preguntas que presentan un escenario de conduccion concreto y preguntan que debe hacer el conductor. Son el tipo mas complejo y el mas frecuente en el examen DGT (aproximadamente 35-40%). El escenario debe incluir detalles especificos: tipo de via, condiciones meteorologicas, otros vehiculos presentes, hora del dia cuando sea relevante.

**4 levels for SITUACIONAL (locked decision — this type has 4 levels, not 3):**

Level definitions (verbatim from CONTEXT.md):
- Nivel 1 (Facil): "1 variable, aplicacion clara de una regla."
- Nivel 2 (Medio): "2 condiciones combinadas. Un distractor activa el instinto equivocado."
- Nivel 3 (Dificil): "Excepcion a una regla conocida, o conflicto de reglas. Las 3 respuestas son plausibles sin dominar la norma."
- Nivel 4 (Muy Dificil): "Dos reglas parecen entrar en conflicto directo. El alumno debe saber cual prevalece. Todas las opciones son muy plausibles."

Distractor rules:
- Nivel 1: One distractor is clearly wrong (violates an obvious rule). One distractor represents a common incorrect instinct.
- Nivel 2: Both wrong options represent plausible-but-wrong responses to one of the two combined conditions. The student who only processes one condition might choose a distractor.
- Nivel 3: All three options are behaviors that would be correct in similar-but-different scenarios. The correct option is the one that applies specifically to the exception or rule conflict in the scenario.
- Nivel 4: All three options are defensible with partial knowledge. The correct answer requires knowing which rule takes priority when two rules appear to conflict.

Scenario quality rules (add as a dedicated section — these are locked decisions from CONTEXT.md):
- Scenarios must include concrete details: road type (autopista, carretera convencional, urbana), speed when relevant, weather conditions when relevant, other vehicles present, time of day when relevant
- Complexity scales with level: Nivel 1 = 1 variable; Nivel 2 = 2 combined; Nivel 3 = exception or multi-rule; Nivel 4 = rules in apparent conflict
- Focus: situacional is the weakest area of current generation — these rules are the primary quality target

Use CONTEXT.md approved examples for all 4 levels:
- Facil: luces largas / vehiculo en sentido contrario
- Medio: glorieta + vehiculo de emergencias
- Dificil: linea continua + adelantar a ciclista
- Muy Dificil: autobus escolar + semaforo en verde

These are all approved reference examples — use them directly.

Checklist for situacional:
```
## Checklist por nivel

### Nivel 1
- [ ] El escenario tiene exactamente 1 variable de decision
- [ ] La regla que se aplica es directa y no tiene excepciones relevantes en este contexto
- [ ] Al menos 1 opcion incorrecta viola una norma obvia

### Nivel 2
- [ ] El escenario combina exactamente 2 condiciones
- [ ] Al menos 1 distractor es la respuesta correcta si solo se considera 1 de las 2 condiciones

### Nivel 3
- [ ] Las 3 opciones son respuestas que serian correctas en escenarios similares
- [ ] La respuesta correcta depende de conocer la excepcion o saber que regla prevalece
- [ ] Un alumno sin conocimiento exacto no puede descartar ninguna opcion por logica

### Nivel 4
- [ ] Las 2 reglas en aparente conflicto estan claramente presentes en el escenario
- [ ] La opcion correcta requiere saber cual regla prevalece cuando ambas aplican
- [ ] Todas las opciones son muy plausibles incluso para alumnos con buen nivel general
```
